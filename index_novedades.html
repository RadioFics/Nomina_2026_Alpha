📋 Formatos y uso
Hoja requerida: Reporte Final
Fila 4: Encabezados — Col B: Cédula, C: Nombre, D: Cargo, E: Centro Costo, F–N: Tipos de novedad
Fila 5+: Datos de empleados con cantidades de horas

Mapeo de columnas a conceptos:
· F → Hora Extra Diurna 125%
· G → Hora Extra Nocturna 175%
· H → Hora Dominical o Festiva Ordinaria 175%
· I → Recargo Nocturno 35%
· J → Recargo Nocturno Festivo 210%
· K → Hora Extra Diurna Festiva 200%
· L → Hora Extra Nocturna Festiva 250%
· M → Recargo Dominical Compensado 0.75%
· N → Hora Dominical No Compensado 175% (se suma con col H si ambas tienen valor)

Instrucciones de instalación
1. Descargar script SQL: schema_novedades.sql
2. Ejecutar en SQL Server Management Studio o Azure Data Studio
3. Configurar connection string en el archivo appsettings.json
4. Instalar dependencia: npm install mssql
5. Usar el botón "Exportar ADECCO" para generar el Excel de novedades del período
